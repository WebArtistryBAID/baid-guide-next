---
title: 作息时间
---

北京中学国际部 2025 - 2026 学年的作息时间如下:

| 时间            | 活动      | 备注                 |
|---------------|---------|--------------------|
| 7:00 - 7:40   | 早餐      | 自愿用餐，有称重自取和套餐可选    |
| 7:00 - 7:55   | 入校，早自习  | 各年级在不同时间入校         |
| 8:00 - 8:40   | 第一课时    | 课时的英文是 period      |
| 8:50 - 9:30   | 第二课时    |                    |
| 9:40 - 10:20  | 第三课时    |                    |
| 10:30 - 11:10 | 第四课时    |                    |
| 11:20 - 12:00 | 第五课时    | 八年级、九年级、十年级、十二年级学生 |
| 11:20 - 12:20 | 第五课时    | 十年级一班、十年级二班、十一年级学生 |
| 11:10 - 13:20 | 午餐      | 有称重自取和套餐可选         |
| 13:20 - 14:00 | 第六课时    |                    |
| 14:10 - 14:50 | 第七课时    |                    |
| 15:00 - 16:00 | 第八课时    | 均为体育               |
| 16:00         | 放学      |                    |
| 16:10 - 16:50 | 第九课时    | 社团、选修课、活动等         |
| 17:00 - 17:40 | 第十课时    | 社团、选修课、活动等         |
| 17:00 - 19:00 | 晚餐      | 自愿用餐，有称重自取和套餐可选    |
| 19:00 - 20:30 | 晚自习第一课时 | 自愿参加，无授课安排         |
| 20:40 - 22:00 | 晚自习第二课时 | 仅住宿生可参加，走读学生必须离校   |
| 22:00 - 22:25 | 住宿生洗漱   |                    |
| 22:30         | 熄灯      |                    |

<Callout>
    <EnrollmentGate when="typical">
        并不是每一个课时都有课程—每个人的课表都不一样。十年级的课程较多，可能每一个课时都填满了课程；但从十一年级开始，自习时间就会增加。如果你选择了 4 门 AP 课程，则课表仍会是基本填满的。但如果你申请了综合英语课程的 [自修](/docs/procedures/self-study) 或选择的 AP 课程更少，则会有更多自习时间。
    </EnrollmentGate>

    <EnrollmentGate when="1+3,0.5,1.5">

    </EnrollmentGate>

    </EnrollmentGate>
</Callout>

<Callout>
    由于学校住宿空间有限，只对居住较远的同学提供住宿。北京中学东坝北校区的住宿为六人间。不同的同学对住宿环境体验不同，但一般认为住宿环境较好。
</Callout>
